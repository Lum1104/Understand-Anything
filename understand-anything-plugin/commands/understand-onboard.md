description: Generate an onboarding guide from an Understand Anything knowledge graph
---

Use the existing Understand Anything onboarding skill.

## Usage

Examples:
- `:understand-onboard`

## Execute

1. Ensure Understand Anything is installed (the universal plugin root should exist):
   - `$HOME/.understand-anything-plugin`

2. Read and follow the workflow in:
   - `$HOME/.understand-anything-plugin/skills/understand-onboard/SKILL.md`
