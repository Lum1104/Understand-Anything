---
name: understand-domain
description: Extract business domain knowledge and generate a domain flow graph
---

Use the existing Understand Anything domain skill.

## Usage

Examples:
- `:understand-domain`
- `:understand-domain --full`

## Execute

1. Ensure Understand Anything is installed (the universal plugin root should exist):
   - `$HOME/.understand-anything-plugin`

2. Read and follow the workflow in:
   - `$HOME/.understand-anything-plugin/skills/understand-domain/SKILL.md`

3. If the user invoked this command with `--full`, interpret that as the `--full` flag referenced by the skill.
