name: understand-knowledge
description: Analyze a Karpathy-pattern LLM wiki and generate an interactive knowledge graph
---

Use the existing Understand Anything knowledge-base skill.

## Usage

Examples:
- `:understand-knowledge`
- `:understand-knowledge /path/to/wiki`

## Execute

1. Ensure Understand Anything is installed (the universal plugin root should exist):
   - `$HOME/.understand-anything-plugin`

2. Read and follow the workflow in:
   - `$HOME/.understand-anything-plugin/skills/understand-knowledge/SKILL.md`

3. If the user provided a directory path after the command, treat it as the target wiki directory.
